# Council Template (ペルソナ会議テンプレート)

## 目的

複数のAIペルソナを使った意思決定・アイデア検証のための会議フレームワーク。
一人創業者が多角的な視点で重要な判断を下すための手順書。

## What is the Council?

「評議会（Council）」は、異なる専門性・視点を持つペルソナが
一つの問いに対して議論する形式の思考実験。

ユーザー、投資家、エンジニア、デザイナー、競合他社…
それぞれの視点から問いを検討することで、盲点を減らす。

## 会議の構造

```
1. 問いの設定（5分）
   - 今日の議題を1文で明確にする

2. ペルソナの選定（2分）
   - 今回の問いに関係するペルソナを3〜5人選ぶ

3. 各ペルソナの発言（各3〜5分）
   - そのペルソナの立場から、正直に意見を述べる

4. 反論フェーズ（5分）
   - 他のペルソナからの反論・懸念を出す

5. 統合・判断（5分）
   - 創業者として、どう判断するかを決める
   - 判断は governance/DECISION_LOG.md に記録
```

## 標準ペルソナ一覧

| ペルソナ | 視点 | 重視すること |
|---------|------|------------|
| パワーユーザー | 使い手 | 機能・使いやすさ |
| 懐疑的なユーザー | 未採用者 | 信頼・障壁 |
| 投資家 | 事業性 | スケール・ROI |
| エンジニア | 技術 | 実装コスト・技術的負債 |
| 競合CEO | 市場 | 差別化・攻撃ポイント |
| 未来の自分（3年後） | 長期 | 持続性・後悔しないか |

## 使い方

1. 議題を決める
2. 関連するペルソナを選ぶ
3. 上記の会議構造に従って進める
4. 判断を `governance/DECISION_LOG.md` に記録
