# Template Business OS

## 目的

テンプレート販売事業の固有OSを管理する。
Notionテンプレート・Figmaテンプレート・コードテンプレート等の
デジタルコンテンツ販売に特化した運営基準。

## 何を保存するか

- **テンプレート品質基準** — リリース前チェックリスト・品質の定義
- **価格設計** — テンプレートの価格帯・バンドル・セール方針
- **配布チャネル戦略** — どこで売るか・どう発見されるか
- **アップデートポリシー** — 購入者への更新・サポートの範囲
- **コンテンツ保護方針** — 著作権・コピー対策
- **コミュニティ連携** — ユーザーコミュニティでの活用促進

## テンプレート品質チェックリスト（最低基準）

- [ ] 使い方ドキュメントが付属している
- [ ] サンプルデータが含まれている（または削除手順がある）
- [ ] モバイルで表示確認済み
- [ ] 1週間以上自分で使ってテスト済み
- [ ] 価格と提供価値のバランスが取れている
