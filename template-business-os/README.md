# Template Business OS

## 目的

テンプレート販売事業の固有OSを管理する。
Notionテンプレート・Figmaテンプレート・コードテンプレート・ドキュメントテンプレート等の
デジタルコンテンツ販売に特化した運営基準。

**このOSは `business-os/` の汎用基準を、テンプレート事業向けに特化させたもの。**

**このOSが答える問い:**
- テンプレートをどう設計し、どう品質担保するか
- いくらで売るか、どこで売るか
- 購入者にどこまでサポートするか
- テンプレートをいつアップデートし、どう通知するか

---

## 責務

| 責務 | 内容 |
|------|------|
| テンプレート品質基準 | リリース前チェックリスト・品質の定義 |
| 価格設計 | テンプレートの価格帯・バンドル・セール方針 |
| 配布チャネル戦略 | どこで売るか・どう発見されるか |
| アップデートポリシー | 購入者への更新・サポートの範囲と告知方法 |
| コンテンツ保護方針 | 著作権・無断転載・コピー対策 |
| サポート範囲の定義 | テンプレートCS対応の範囲（無制限にしない）|

## 他OSとの関係

| OS | 関係 |
|----|------|
| `business-os/` | 収益モデル・GTMはbusiness-osの汎用基準を継承 |
| `brand-os/` | テンプレートのコピー・デザインはbrand-osに従う |
| `legal-os/` | テンプレートの利用規約・著作権方針はlegal-osと整合 |
| `customer-success-os/` | テンプレート購入者のサポートはcustomer-success-osの基準を適用 |
| `templates/` | このOSで定義した基準を `templates/` に実装する |
