# Business OS

## 目的

事業戦略・収益モデル・GTM（Go-to-Market）・価格設計の共通思想と手順を定義する。
各プロダクトのビジネス判断の土台となるOS。

## 何を保存するか

- **事業モデルの評価基準** — 何を収益化するか・しないかの判断軸
- **GTMフレームワーク** — 新プロダクトをどう世に出すかの手順
- **価格設計原則** — 価格帯・プラン構成・値上げのルール
- **競合分析フレームワーク** — 市場をどう見るかの視点
- **プロダクト-マーケットフィット基準** — PMFをどう定義・測定するか
- **撤退基準** — いつプロダクトを終了するかの判断軸

## 事業モデル原則

1. **サブスクリプション優先** — 一括販売より継続課金を優先設計
2. **フリーミアムは慎重に** — 無料ユーザーのコストを正確に計算する
3. **価格は後から上げる** — 低価格で始め、価値提供後に引き上げる
4. **B2C + B2B2C の二段構え** — 個人ユーザーと法人チャネルを並走

## 今後の再利用計画

- 各新プロダクト立ち上げ時に `BUSINESS_STRATEGY_TEMPLATE.md` を使用
- GTMプレイブックをプロダクトタイプ別に蓄積
