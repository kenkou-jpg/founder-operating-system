# Finance OS

## 目的

財務管理・コスト構造・投資判断の基準を定義する。
一人創業者が財務的に持続可能な事業を複数運営するための手順書。

## 何を保存するか

- **コスト管理方針** — 固定費・変動費の管理ルール、上限基準
- **投資判断フレームワーク** — 何にお金をかけるか・かけないかの軸
- **収益目標の設定方法** — ARR/MRR目標の設定と追跡
- **財務レビューテンプレート** — 月次・四半期・年次のレビュー手順
- **税務・会計の運営ルール** — 経費計上・確定申告の準備手順
- **キャッシュフロー管理** — ランウェイの計算と管理

## 財務原則

1. **ランウェイを常に把握する** — 最低12ヶ月のランウェイを維持
2. **コストは意図的に使う** — 「なんとなく」の支出を排除
3. **収益より先にコストを増やさない** — 収益が証明されてから固定費を増やす
4. **ツールコストを四半期ごとに見直す** — 使っていないSaaSを切る

## 月次財務チェック項目

- [ ] MRR / ARR の確認
- [ ] 月次固定費の確認
- [ ] チャーン率の確認
- [ ] ランウェイの再計算
- [ ] 未払い・未受領の確認
