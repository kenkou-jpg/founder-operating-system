# FOUNDER_EFFICIENCY

> Founder Asset の週次増加を管理する。
> 1週間で何種類の Asset が増えたかを Founder Efficiency として定量化する。

---

## Asset カテゴリ定義

| カテゴリ | 内容 | カウント例 |
|---------|------|---------|
| **PR** | 完了した PR 数 | PR-043 完了 → +1 |
| **Template** | 新規作成・更新したテンプレート | PR Input Sheet 更新 → +1 |
| **Prompt** | 新規追加・改善した AI プロンプト | PR Prompt Template 更新 → +1 |
| **Automation** | 追加した自動化ルール・スクリプト | GitHub Actions 追加 → +1 |
| **Knowledge** | `knowledge-os/` に記録した知見 | ユーザーインタビュー記録 → +1 |
| **Brand** | note 記事・X 投稿・LP コンテンツ | note 記事公開 → +1 |
| **Research** | `research-os/` に記録した調査インサイト | 競合調査まとめ → +1 |
| **OS Improvement** | Founder OS へのフィードバック・改善 | Analytics OS 追加 → +1 |
| **Decision** | Decision Log に記録した Founder 判断 | DLOG-006 記録 → +1 |
| **Capability** | 新たに習得・強化した Founder Capability | SEO 知識を IPPO に適用 → +1 |
