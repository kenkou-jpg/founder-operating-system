## 利用カウント基準

| 行動 | カウント |
|-----|---------|
| その OS のファイルを参照した | +1 |
| その OS のテンプレートを使用した | +1 |
| その OS のルールに従って判断した | +1 |
| 同じ OS を同一タスク内で複数回参照 | 1回のみカウント |
