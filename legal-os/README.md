# Legal OS

## 目的

全プロダクト共通の法務方針・利用規約ひな形・契約テンプレートを管理する。
法務コストを最小化しつつ、ユーザー保護と事業保護を両立する。

## 何を保存するか

- **利用規約ひな形** — プロダクト別にカスタマイズできる基本テンプレート
- **プライバシーポリシーひな形** — データ収集・利用・保護の標準文書
- **特定商取引法表記** — EC・サブスクリプションサービス向け
- **免責事項テンプレート** — 健康・農業等のセンシティブ領域向け
- **外部サービス利用規約チェックリスト** — Supabase・Expo等の利用条件確認
- **著作権・ライセンス方針** — OSSライセンスの扱い・自社著作物の保護

## 法務原則

1. **ユーザーに読まれる文書を書く** — 難解な法律用語より平易な日本語
2. **データ最小化** — 必要なデータだけ収集する、それが最大の法務リスク低減
3. **変更は事前告知** — 規約変更は30日前にユーザーへ通知
4. **記録を残す** — 同意取得・通知の記録は保持

## 今後の再利用計画

- 各プロダクト立ち上げ時にひな形をカスタマイズして使用
- ヘルスケア（Fasting App）・農業（AgriPath）の特殊規制に対応するバリアント作成
